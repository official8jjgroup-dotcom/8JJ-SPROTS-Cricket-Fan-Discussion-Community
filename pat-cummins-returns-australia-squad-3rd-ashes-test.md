# Pat Cummins Returns: Australia Squad Announced for 3rd Ashes Test

Pat Cummins is back in the spotlight as Australia announces its 15-man squad for the third Ashes Test against England in Adelaide. The return of Cummins marks a significant boost for the hosts, who hold a 2-0 lead in the five-match series. The third Test, scheduled to start at Adelaide Oval on December 17, is set to be a pivotal contest in this highly anticipated series.
